# Noticeable Angular Sample

This project shows how to integrate Noticeable Widgets in an [Angular](https://angular.dev/) Web app.

The repository's commit history walks through the integration step by step:
https://github.com/noticeable-app/noticeable-sample-angular/commits/main

## Stack

- Angular 21 (standalone components)
- Vite + esbuild via `@angular/build`
- TypeScript

## Run locally

```sh
npm install
npm start
```

Then open `http://localhost:4200/`.
